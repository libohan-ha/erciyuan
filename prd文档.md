### **个人二次元图片收藏系统 - PRD**

#### **1. 项目概述**

本项目是一个个人使用的、私有的二次元图片在线收藏与管理网站。用户可以方便地上传、分类、打标签、搜索和浏览自己收藏的图片，旨在打造一个功能强大、界面美观的个人图片库。

#### **2. 用户角色与故事**

- **用户角色**：系统所有者（唯一用户）。
- **用户故事**：
  - 作为一个二次元爱好者，我希望能快速上传我喜欢的图片，以便随时在线查看。
  - 作为一个收藏者，我希望能给图片打上标签（如角色名、作品名），方便我按主题筛选。
  - 作为一个整理者，我希望能创建不同的相册，将图片分门别类地存放。
  - 作为一个使用者，我希望能通过关键词快速搜索到我想要的任何一张图片。

#### **3. 功能需求**

**3.1 用户认证模块**

- **FR-1.1 注册**：提供用户名和密码即可完成注册。
- **FR-1.2 登录**：输入正确的用户名和密码，系统返回JWT Token用于后续认证。
- **FR-1.3 登出**：清除本地Token，退出登录状态。
- **FR-1.4 个人资料管理**：在个人中心可以修改密码和更换头像。

**3.2 图片管理模块**

- **FR-2.1 图片上传**：提供一个表单，允许用户选择本地图片文件，并填写标题、描述、标签，选择所属相册后上传。
- **FR-2.2 图片浏览（画廊）**：首页以瀑布流形式展示用户上传的所有图片。
- **FR-2.3 图片详情**：点击任意图片，进入详情页，查看大图、标题、描述、标签等信息。
- **FR-2.4 图片编辑**：在详情页或管理界面，可以修改图片的标题、描述、标签和所属相册。
- **FR-2.5 图片删除**：可以删除任何一张图片，删除后图片记录和文件本身都应被移除。

**3.3 相册管理模块**

- **FR-3.1 创建相册**：用户可以创建新的相册，并设置名称和描述。
- **FR-3.2 查看相册**：在个人中心可以查看所有已创建的相册列表。
- **FR-3.3 管理相册**：可以编辑相册名称、描述，或删除整个相册。删除相册不影响图片本身，只是解除图片与该相册的归属关系。

**3.4 搜索与筛选模块**

- **FR-4.1 关键词搜索**：在画廊页面提供一个搜索框，可以根据图片标题或标签中的关键词进行搜索。
- **FR-4.2 标签筛选**：在画廊侧边栏展示所有标签云，点击任意标签，即可筛选出所有包含该标签的图片。
- **FR-4.3 相册筛选**：在画廊侧边栏展示所有相册，点击任意相册，即可筛选出该相册内的所有图片。

#### **4. 非功能需求**

- **NFR-1 技术栈**：
  - 前端：HTML, CSS, JavaScript (原生)
  - 后端：Node.js, Express.js
  - 数据库：MongoDB
  - 接口：RESTful API
- **NFR-2 设计风格**：
  - UI组件库：MUI (Material Web Components)
  - CSS框架：Tailwind CSS
  - 整体风格：现代、简洁、清爽。
- **NFR-3 性能**：图片加载应流畅，瀑布流布局不应造成明显卡顿。
- **NFR-4 安全性**：用户密码必须加密存储；所有需要权限的API接口必须通过JWT验证。
